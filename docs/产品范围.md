# 服务范围

## 名称

暂定：Trading Agent Assistant 本地投研服务包

## 一句话定位

基于开源 TradingAgents 生态，为中文 Windows 用户提供本地安装、模型配置、数据源接入、投研报告、使用培训和售后支持服务。

## 服务对象

- 不想手动配置 Python、Git、Docker 的投资研究用户
- 希望用中文模型做个股分析和策略研究的用户
- 需要 A股、港股、美股统一入口的量化学习用户
- 需要本地保存配置、报告和诊断信息的个人用户

## 服务内容

- Windows 本地部署和启动入口配置
- 中文网页配置页
- 大模型 API Key / Base URL 配置
- 数据源连通性测试
- A股、港股、美股示例验收
- 中文投研报告整理
- 历史报告和 Word/PDF 导出
- 系统诊断和售后排查
- 客户使用培训和常见问题支持

## 开源边界

- 核心研究框架参考并集成开源 TradingAgents 项目
- 我们不销售开源软件本体
- 我们交付的是部署、配置、中文化、培训和售后服务
- 客户应遵守相关开源许可证、模型服务条款和数据源使用条款

## 数据市场

- A股：Tushare Pro、Akshare，后续可评估 Baostock
- 港股：Akshare、Yahoo/yfinance，后续可评估富途、长桥、老虎
- 美股：Yahoo/yfinance、Alpha Vantage，后续可评估 IBKR、Alpaca

## 明确边界

- 首版只做研究、学习、回测和辅助分析
- 不默认开放实盘自动交易
- 不构成证券投资咨询
- 不保证数据完整、准确或实时
- 不保证模型输出正确
- 不提供收益承诺
