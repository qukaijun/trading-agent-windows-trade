# 客户交付验收单

## 安装验收

- [ ] 客户电脑为 Windows 10/11 64 位系统。
- [ ] 安装包可正常打开，安装向导为中文界面。
- [ ] 安装过程中自动完成本地运行环境安装。
- [ ] 客户不需要手动安装 Python、Git、Docker。
- [ ] 客户不需要打开命令行或输入脚本命令。
- [ ] 桌面存在“Trading Agent 投研工作台”快捷方式。
- [ ] 开始菜单存在中文入口。
- [ ] 双击桌面图标后，浏览器可打开 `http://127.0.0.1:8787/`。

## 配置验收

- [ ] 可在首页“模型配置”中选择大模型供应商。
- [ ] 可填写模型名称、Base URL、API Key。
- [ ] 可保存模型配置。
- [ ] 可测试模型连接。
- [ ] API Key 不在页面明文回显。
- [ ] 当前模式显示为“投研分析 / 模拟验证”。

## 市场验收

- [ ] A 股样例可识别：`600519`。
- [ ] 港股样例可识别：`HK:00700.HK`。
- [ ] 美股样例可识别：`US:AAPL`。
- [ ] 美股客户样例可识别：`PDD` 和 `US:PDD`。
- [ ] 数据源检查可返回明确结果。
- [ ] 美股数据源失败时，页面显示中文原因和下一步，不暴露英文异常、JSON、endpoint 或 traceback。

## 投研报告验收

- [ ] 可发起 A 股样例分析。
- [ ] 可发起港股样例分析。
- [ ] 可发起美股样例分析。
- [ ] 分析失败时页面有明确中文提示。
- [ ] 模型未配置或模型连接失败时，页面阻止无意义分析并提示先配置/测试模型。
- [ ] 分析完成后，页面优先展示分区中文报告。
- [ ] 报告包含摘要、标的信息、行情数据、模型结论、风险和使用边界。
- [ ] 历史报告可重新打开。
- [ ] 报告可下载 Markdown。
- [ ] 报告可下载 Word。
- [ ] 报告可下载 PDF。

## 售后验收

- [ ] “停止本地服务”入口可正常执行。
- [ ] “修复本地运行环境”入口可正常执行。
- [ ] “导出售后诊断日志”入口可生成压缩包。
- [ ] 首页“系统诊断”可刷新。
- [ ] 首页“系统诊断”可导出诊断文本。
- [ ] 首页“关于本软件”显示开源声明、服务范围和风险边界。
- [ ] 客户已收到风险提示与免责声明。

## 交付结论

客户确认当前版本用于投研分析、策略研究、学习和模拟验证，不开放实盘自动交易。客户理解软件主体基于开源项目集成，服务方交付的是部署、配置、培训、运维、诊断和售后支持服务。
