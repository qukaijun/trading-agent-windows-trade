# 大模型配置指南

## 客户配置方式

1. 双击桌面图标“Trading Agent 投研工作台”。
2. 在首页左侧打开“模型配置”。
3. 选择模型供应商。
4. 填写模型名称、Base URL 和 API Key。
5. 点击“保存配置”。
6. 点击“测试模型”，确认连接可用。

API Key 不会在页面明文回显。输入框留空时，系统会保留原有 API Key。

## 常用供应商

- 通义千问兼容 OpenAI 接口。
- DeepSeek 兼容 OpenAI 接口。
- OpenAI 官方接口。
- OpenRouter 或客户自有 OpenAI 兼容网关。
- 本地或私有化 OpenAI 兼容接口。

当前 Windows MVP 的网页配置入口不直接提供 Google/Gemini 官方 API 配置。Gemini 如需接入，应通过客户已有的 OpenAI 兼容网关或作为定制接入评估，不作为当前标准承诺。

## 常见问题

- 测试失败时，先检查 API Key 是否正确。
- 再检查 Base URL 是否能访问。
- 再检查模型名称是否与供应商后台一致。
- 国内网络环境下，优先使用国内可访问的模型供应商或客户自己的私有化网关。

## 售后工程师备用方式

客户日常不需要使用命令行。售后工程师远程排查时，可使用安装目录中的备用配置脚本，但不把该方式作为客户说明。
