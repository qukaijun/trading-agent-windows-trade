# 风险提示与免责声明

Trading Agent Assistant 当前版本仅用于投研分析、策略研究、学习和模拟验证。

## 开源软件与服务交付说明

本地工具基于开源 TradingAgents 生态进行中文化、本地部署和服务交付。服务方提供的是安装配置、模型/数据源接入、培训、运维和售后支持服务，不销售开源软件本体。

客户应自行遵守相关开源许可证、第三方模型服务条款、数据源使用条款和所在地适用法律法规。

## 不构成投资建议

软件输出的分析、评级、方向判断、风险提示和策略信号均不构成证券、基金、期货、外汇或任何金融产品的投资建议。

## 不承诺收益

软件不保证任何分析结果正确，不保证任何交易或投资收益，也不承诺降低亏损。

## 数据风险

第三方数据源可能存在延迟、缺失、错误、限流或服务中断。免费数据源不适合被包装为实盘级行情。

## 模型风险

大模型可能产生幻觉、遗漏关键信息、误解财务数据或给出不稳定结论。客户应结合自身判断和其他信息源独立决策。

## 当前版本边界

首版不开放实盘自动交易，不托管资金，不保管券商账户，不代客户下单。
