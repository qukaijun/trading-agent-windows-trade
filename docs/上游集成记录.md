# 上游集成记录

验证日期：2026-06-06

## 原版 TradingAgents

当前已拉取：

```text
vendor/upstream/TradingAgents
```

本机检查到的上游信息：

- 包名：`tradingagents`
- 版本：`0.2.5`
- Python 要求：`>=3.10`
- CLI 入口：`tradingagents = cli.main:app`
- 配置覆盖：支持 `TRADINGAGENTS_*` 环境变量
- 默认数据源：`yfinance`
- 支持 A 股 Yahoo Finance 后缀：上海 `.SS`，深圳 `.SZ`
- 支持港股 Yahoo Finance 后缀：`.HK`

## 对本项目的影响

`scripts/configure.ps1` 必须写入原版识别的变量：

```text
TRADINGAGENTS_LLM_PROVIDER
TRADINGAGENTS_DEEP_THINK_LLM
TRADINGAGENTS_QUICK_THINK_LLM
TRADINGAGENTS_OUTPUT_LANGUAGE
TRADINGAGENTS_MAX_DEBATE_ROUNDS
TRADINGAGENTS_MAX_RISK_ROUNDS
TRADINGAGENTS_TEMPERATURE
```

本项目自己的产品变量使用 `TRADINGAGENT_*` 前缀，例如：

```text
TRADINGAGENT_MARKETS=A,HK,US
TRADINGAGENT_ENABLE_LIVE_TRADING=0
```

## 中文参考仓库

`TradingAgents-CN` 只作为参考，不作为首版直接运行底座。原因：

- 架构较重
- 更适合 Web 全栈部署
- Windows 一键安装成本更高

首版先对接原版 TradingAgents，并在外层补中文配置、市场代码规范、安装器和数据源策略。
