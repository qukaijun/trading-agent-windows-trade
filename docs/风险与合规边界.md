# 风险与合规边界

## 产品声明

Trading Agent 助手首版仅用于投研、学习、策略原型、回测和辅助分析，不构成证券投资咨询、投资建议、收益承诺或自动交易服务。

## 必须提示用户

- 金融市场存在风险，历史表现不代表未来收益
- AI 输出可能错误、遗漏或过度推断
- 第三方数据源可能延迟、缺失或不准确
- 免费行情源通常不适合实盘、高频或强实时场景
- 用户自行承担投资决策和 API 调用费用

## 不应出现的营销表述

- 稳赚
- 自动赚钱
- 保本
- 买卖点神器
- 实盘托管
- 代客理财
- 荐股服务
