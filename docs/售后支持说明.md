# 售后支持说明

## 支持范围

- 安装失败排查。
- 启动失败排查。
- 大模型 API Key 配置排查。
- 数据源连通性排查。
- 日志导出和问题定位。
- 版本更新说明。

## 不包含

- 证券投资咨询。
- 收益承诺。
- 代客户操作证券账户。
- 免费数据源稳定性担保。
- 客户自有网络、代理、防火墙或杀毒软件导致的问题兜底。

## 客户需要提供的信息

- Windows 版本。
- 问题截图。
- 使用的大模型供应商。
- 输入的股票代码。
- 首页“系统诊断”面板截图。
- 通过“导出售后诊断日志”生成的日志压缩包。

客户不需要提供 API Key 明文，也不需要打开命令行。

## 推荐响应流程

1. 让客户双击桌面图标，确认能否打开首页。
2. 让客户点击首页左侧“系统诊断 -> 刷新诊断”。
3. 让客户截图诊断结果，或点击“导出诊断”。
4. 确认模型配置是否存在。
5. 确认数据源检查结果。
6. 如果本地服务无法启动，让客户从开始菜单点击“导出售后诊断日志”。
7. 必要时让客户从开始菜单点击“修复本地运行环境”，完成后重新启动。

## 诊断接口

```text
GET /api/diagnostics
GET /api/diagnostics.txt
```

这些接口供售后人员排查使用。客户侧优先使用页面按钮和开始菜单中文入口。
