# 数据源配置指南

| 数据源 | 市场 | 状态 | 说明 |
| --- | --- | --- | --- |
| Yahoo Finance / yfinance | A股、港股、美股 | 已集成 | 美股和港股基础行情较方便，A股作为补充 |

## 报告中的行情快照

当前报告会先使用 yfinance 生成一段确定性行情快照，包括：

- 最新交易日
- 最新收盘价
- 单日涨跌幅
- 近 5 日涨跌幅
- 近 20 日涨跌幅
- 近 20 日最高/最低
- 最新成交量
- 近 20 日平均成交量

这部分用于提供数据底座，不等同于模型结论，也不构成买卖建议。

## Tushare Token

首次配置向导会询问可选的 Tushare Token。也可以手动写入：

```text
%USERPROFILE%\.trading-agent-assistant\.env
```

示例：

```text
TUSHARE_TOKEN=你的token
```

## 数据源注意事项

- 免费数据源可能延迟、缺失或限流。
- 不同数据源对同一股票的字段口径可能不同。
- 首版不把免费行情源包装成实盘级行情。
- 客户交付时应先用 A股、港股、美股各一个样例完成验收。
