# 快速开始

## 启动

从桌面或开始菜单点击：

```text
Trading Agent 投研工作台
```

启动成功后，浏览器会自动打开：

```text
http://127.0.0.1:8787/
```

如果浏览器没有自动打开，可以手动访问以上地址。

## 股票输入示例

```text
600519
A:600519.SH
HK:00700.HK
HK:9988
PDD
US:PDD
US:AAPL
NVDA
```

## 基础流程

1. 在页面左侧确认“模型配置”已保存。
2. 输入股票代码或股票名称。
3. 点击“识别市场”，确认系统识别为 A 股、港股或美股。
4. 点击“检查数据源”，确认行情源可访问。
5. 点击“生成投研报告”，生成 TradingAgents 投研报告。
6. 在页面右侧直接阅读分区报告。
7. 如需归档，再点击“下载 Markdown”“下载 Word”或“下载 PDF”。

## 报告查看

分析完成后，页面会优先展示整理后的中文投研报告。报告按摘要、标的信息、确定性行情数据、模型结论、核心依据、主要风险和使用边界分区展示。

下载功能是附带能力，核心使用方式是在页面内直接阅读和复核报告。

## 常见问题

如果分析失败，优先检查：

- API Key 是否正确。
- Base URL 是否可访问。
- 模型名称是否存在。
- 当前网络是否能访问模型服务和数据源。
- 股票代码是否被正确识别。
- 美股数据源依赖第三方公开行情源，客户网络、第三方限制或临时连接失败都可能导致 `PDD` / `US:PDD` 检查失败。

仍无法解决时，从开始菜单点击“导出售后诊断日志”，把生成的 `support-logs` 压缩包发给服务人员。
