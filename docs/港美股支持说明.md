# 港美股支持说明

## 支持结论

当前版本支持港股和美股的投研分析、市场识别和数据源检查。

## 港股输入

```text
HK:00700.HK
HK:9988
00700.HK
9988
腾讯控股
阿里巴巴
```

系统会归一化为：

```text
00700.HK
09988.HK
```

## 美股输入

```text
US:AAPL
US:NVDA
PDD
US:PDD
AAPL
NVDA
苹果
英伟达
微软
```

系统会归一化为：

```text
AAPL
NVDA
PDD
MSFT
```

## 数据源

- 港股：当前优先使用第三方公开行情源和 Akshare 作为基础检查。
- 美股：当前优先使用第三方公开行情源做基础行情检查，`PDD` / `US:PDD` 可识别为美股代码。
- 如客户网络无法访问第三方公开行情源，或数据源临时限流/失败，页面会显示中文状态并提示导出售后诊断包。该风险不视为已彻底解决，需要在客户真实网络环境继续验证，或后续拍板备用数据源。

## 重要边界

港美股投研分析和港美股实盘交易是两件事。当前版本只交付投研分析和模拟验证，不开放真实券商账户自动下单。
