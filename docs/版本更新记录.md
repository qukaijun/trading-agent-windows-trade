# 版本更新记录

## 0.1.0-mvp

当前版本定位为 Windows 本地 AI 投研分析助手，用于投研分析、策略研究、学习和模拟验证。

本项目基于开源 TradingAgents 生态做中文化、本地部署和服务交付。我们提供的是安装配置、模型/数据源接入、培训、运维和售后支持服务，不销售开源软件本体。

### 已包含

- Windows 本地服务和一键安装包结构
- 网页模型配置
- 模型连接测试
- A股、港股、美股代码识别
- 数据源检查
- 行情快照
- 中文投研报告整理
- 历史报告本地保存
- Markdown / Word / PDF 导出
- 系统诊断和诊断文本导出
- 关于本软件和开源服务说明
- MT5 模拟盘信号桥接预留

### 明确不包含

- 实盘自动交易
- 托管资金或券商账户
- 证券投资咨询
- 收益承诺
- 免费数据源实时性或完整性担保
- 对开源软件本体的独占销售或闭源授权

### 交付建议

服务交付时建议完成：

- 模型配置和测试模型
- 服务方/售后联系方式配置
- A股样例识别和分析
- 港股样例识别和分析
- 美股样例识别和分析
- 报告导出 Word/PDF
- 系统诊断截图
- 开源声明、风险提示与免责声明确认
