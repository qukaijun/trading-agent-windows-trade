# Windows 一键安装包方案

## 安装目标

普通 Windows 10/11 64-bit 用户下载安装包后，可以完成：

1. 安装内置 Python 运行时
2. 安装离线依赖
3. 运行中文配置向导
4. 启动本地 Trading Agent 服务
5. 自动打开浏览器页面

## 推荐技术

- 安装器：Inno Setup
- 运行时：Python 3.12 或 3.13，需根据上游 TradingAgents 依赖验证后固定
- 离线依赖：`wheelhouse/`
- 启动脚本：PowerShell
- 本地服务：FastAPI 或上游自带 CLI/API 包装
- 日志：`logs/`

## 包内目录

```text
{app}/
  scripts/
  agent_host/
  vendor/upstream/TradingAgents/
  wheelhouse/
  runtime/python-installer/
  docs/
  legal/
  logs/
```

## 首版脚本

- `fetch-upstream.ps1`：拉取或更新参考上游源码
- `bootstrap-dev.ps1`：开发机创建虚拟环境并安装依赖
- `configure.ps1`：中文配置向导
- `start.ps1`：启动本地服务
- `stop.ps1`：停止服务
- `export-logs.ps1`：导出诊断日志
- `build-installer.ps1`：调用 Inno Setup 构建安装包

## 验收标准

- 干净 Windows 10/11 可安装
- 不依赖用户手动安装 Git、Python、Node、Docker
- 安装过程中文可读
- 启动失败时能看到中文错误和日志位置
- 至少完成一个 A 股、一个港股、一个美股的示例分析
