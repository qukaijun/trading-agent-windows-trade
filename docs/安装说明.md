# 安装说明

## 适用环境

- Windows 10 / Windows 11 64 位系统。
- 普通用户权限即可安装。
- 不要求客户预装 Python、Git、Docker 或打开命令行。
- 需要能访问所配置的大模型 API 和行情数据源。

## 客户安装流程

1. 双击运行 `TradingAgentAssistant-Setup-0.1.0-mvp.exe`。
2. 按中文安装向导点击“下一步”和“安装”。
3. 安装过程中会自动安装本地运行环境，可能需要几分钟，请不要关闭安装窗口。
4. 安装完成后，可勾选“启动 Trading Agent 投研工作台”，或双击桌面图标。
5. 浏览器会自动打开 `http://127.0.0.1:8787/`。
6. 在页面左侧“模型配置”中填写模型供应商、模型名称、Base URL 和 API Key。
7. 保存配置后，输入股票代码，先“识别市场”，再“检查数据源”，最后“生成投研报告”。

## 常用入口

- `启动 Trading Agent 投研工作台`：打开本地投研分析页面。
- `停止本地服务`：关闭本地后台服务。
- `修复本地运行环境`：自动重装 Python 运行环境和依赖。
- `导出售后诊断日志`：生成诊断压缩包，便于发给服务人员。
- `打开使用文档`：打开本地文档目录。
- `打开 MT5 模拟盘 EA`：打开 MT5 模拟盘 EA 文件目录。

## 配置和隐私

客户的 API Key 保存在本机：

```text
%USERPROFILE%\.trading-agent-assistant\.env
```

请勿把该文件发给他人。售后排查时优先使用“导出售后诊断日志”，诊断日志不会要求客户直接复制 API Key。

## 版本边界

当前版本用于投研分析、策略研究、学习和模拟验证，不开放实盘自动交易。软件主体基于开源项目集成，服务交付内容包括本地化部署、模型和数据源配置、培训、运维和售后支持。
